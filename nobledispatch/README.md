# NobleDispatch — Brand Assets

Public-facing brand graphics for the NobleDispatch platform and ID Hotshot Transportation (founding tenant).

See [`brand/README.md`](brand/README.md) for the role of each asset.

These files are referenced at deploy time by `prompts/99-brand-integration.md` (kept in the private NobleDispatch design repo, not here).

## Raw URLs

Once on `main`, each asset is reachable at:

```
https://raw.githubusercontent.com/desertshibari-crypto/ID-hot-shot/main/nobledispatch/brand/<filename>
```

Files:

- `noble_dispatch_banner.png` — login background hero
- `noble_dispatch_login_reference.png` — login layout reference
- `noble_dispatch_login_empty.png` — login layout reference (backup)
- `dashboard_light_mockup.png` — light-theme dashboard design target
- `dashboard_dark_mockup.png` — dark-theme dashboard design target

## What is NOT in this folder

The full NobleDispatch design package (business documents, deployment prompts, website prototype, license) is intentionally **not** published here — it contains sensitive business documents and is meant for a private repository. See the package's own `PUSH_TO_GITHUB.md` for the proper home.
